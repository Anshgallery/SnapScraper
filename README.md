# Flask Image Scraper

A basic Flask web application that downloads images from Google Image Search using Python, BeautifulSoup, and Requests.

## About the Project

This project demonstrates how to build a simple web-based image scraper using Flask. The application sends a request to Google Images, extracts image tags from the HTML response, downloads the images, and saves them locally inside an `images` folder.

## Features

- Flask-based web application
- Scrapes image tags from a webpage
- Downloads images automatically
- Saves images in a local folder
- Simple route-based structure
- Easy to understand for beginners
